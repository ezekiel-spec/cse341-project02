# CSE 341 - Media Library API

This is a Node.js project for the CSE 341 Web Backend Development II course. It provides a RESTful API to manage a library of movies and actors using Express.js and MongoDB.

## Live Demo
* **Render URL:** [https://cse341-project02-urrq.onrender.com/movies](https://cse341-project02-urrq.onrender.com/movies)
* **API Documentation (Swagger):** [https://cse341-project02-urrq.onrender.com/api-docs](https://cse341-project02-urrq.onrender.com/api-docs)

## Features
- Full CRUD operations for Movies and Actors.
- MongoDB Atlas integration.
- Swagger UI documentation.
- Environment variable security with dotenv.

## Installation
1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file with your `MONGODB_URI`.
4. Run `npm start`.
